# Yuval Fitness Pro MVP

אפליקציית MVP למאמן כושר.

כולל:
- React + Vite
- PWA בסיסי
- יצירת אימון מתבנית
- עריכת סטים / חזרות / משקל / מנוחה
- ספריית תרגילים
- שמירה אוטומטית בדפדפן
- ייצוא PDF
- מותאם מובייל, אייפון ואנדרואיד

## הרצה מקומית

1. התקן Node.js LTS
2. פתח את התיקייה ב-Cursor או VS Code
3. הרץ:

```bash
npm install
npm run dev
```

4. פתח את הכתובת שמופיעה בטרמינל, לרוב:

```bash
http://localhost:5173
```

## בנייה לפרסום

```bash
npm run build
```

## התקנה בטלפון

אחרי העלאה ל-Vercel / Netlify:
- פתח את האתר בדפדפן בטלפון
- iPhone: Share → Add to Home Screen
- Android: Menu → Add to Home screen
